# 八音十六律

## 介绍

八音十六律师一种完全是想象中的音律。其存在基频为$2^i$,$i\in N^{+}$的音波。

## 音律划分

将一个倍频区间作为一个音高区间，然后对音高区间进行划分得到音律。

其中n=16,$a_1$为标准音频率，采用$a_1=512\times2^{-\frac{4}{16}}\approx430.54$Hz。

$$f=a_1 2^{\frac{x}{12}}$$

$$x\in{N^{+}}$$

式中：

$f$为音高对应的频率，Hz

$x$为相对于$a_1$按键的位置，$x$=0得到$a_1$本身频率，$x$=12得到$a_2$频率
