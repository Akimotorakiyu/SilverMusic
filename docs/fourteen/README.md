# 七音十四律

## 介绍

七音十四律。

## 音律划分

将一个倍频区间作为一个音高区间，然后对音高区间进行划分得到音律。

其中n=10,$a_1$为标准音频率。

$$f=a_1 2^{\frac{x}{14}}$$

$$x\in{N^{+}}$$

式中：

$f$为音高对应的频率，Hz；

$x$为相对于$a_1$按键的位置，$x=0$得到$a_1$本身频率，$x=10$得到$a_2$频率。
